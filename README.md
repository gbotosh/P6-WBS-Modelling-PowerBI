# Primavera P6 WBS Modelling in Power BI

This project demonstrates two approaches for modelling Primavera P6 WBS hierarchies in Power BI:

## Methods

### Power Query
- Self-merging approach
- Suitable for Excel exports

### DAX Parent-Child
- Uses PATH() and PATHITEM()
- Suitable when Parent WBS ID is available from a Primavera database or XER import

## Benefits

- Visualise WBS hierarchies
- Filter KPIs by WBS
- Drill down charts by WBS
- Enable decomposition tree analysis

## Files

- PBIX report
- Screenshots

## Dataset

The report uses the Primavera P6 prototype/sample project (Melrose Plant Expansion Project) and is intended to demonstrate techniques for modelling WBS hierarchies in Power BI.
